# Advent of Code 2024 - Starting from Day 1

This repository contains my Python solutions for [Advent of Code 2024, Day 1](https://adventofcode.com/2024/day/1). This is entirely a personal coding journey; and I have limited myself to using only my own knowledge and notes instead of heavily relying on AI tools.

# Foreword
Maybe AI will take over when it comes to writing code, but my bet is that there will always be a need for an experienced human coder. In my own experience working with AI tools in my job, I still can't trust AI code with my eyes closed. We have many integrations, and AI has a limited context window. For example, if there is an error in the past conversation, it keeps repeating the same mistake.

This is critical, in my opinion, because juniors tend to blindly apply what the AI suggests. Sometimes it gives us a delusion of two bad choices where we think we have to choose the least worst one but that is not actually the case. I also argue that for some simple tasks AI produces an overcomplex solution which is beyond our needs.  

Besides, I like to learn and improve my cognitive skills, and I hope these skills will always be needed. It's true that there is a chance to write better code with the help of AI, but one should also practice and learn on their own. My best analogy is chess. People are playing better chess compared to past generations. Because of engines, people have begun to understand and discover new ideas, and the average player is already playing much better. They don't need as much coaching input, and it is way more affordable now. It's the same thing with coding and computer science. It is not an exclusive club anymore.
